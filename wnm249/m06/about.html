About This Project

This website was created as part of a midterm project to showcase the process of designing and developing a user-friendly guide site. It demonstrates planning, user interface design, and responsive web development.

The theme “How to Make a Cappuccino” was chosen for its approachable and relatable topic — bringing together design, structure, and creativity while teaching a fun skill.
